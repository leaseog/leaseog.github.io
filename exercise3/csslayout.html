Cascading Syle Sheets

Provide the style directives to marked-up content.
They determine:

  • color
  • size
  • arrangement
  • and more...



How to add CSS to HTML

3 different ways:

  1. Inline

    < p style="color:blue" >...< /p >


  2. Internal
    < head > < style type="test/css" >
    p { color : blue ; }
    < /style>
    < /head >
    < body>
    < p > ... < /p >
    < /body>

  3. External
    have separate css file and it will be linked on html file.
